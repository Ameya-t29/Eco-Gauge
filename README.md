# Eco-Gauge
Air and Noise Pollution Monitoring Dashboard
EcoGauge is a real-time environmental monitoring dashboard designed to track and analyze air quality and noise pollution levels. It collects data from sensors to measure key parameters such as AQI, PM2.5, PM10, and sound intensity (dB), presenting them through interactive visualizations like charts and maps.

The platform enables users and authorities to monitor pollution trends, receive alerts for critical conditions, and make data-driven decisions to improve environmental health and safety.

Key Features
 Real-time air and noise pollution monitoring
 Interactive dashboard with charts 
 Map visualization
 Smart alerts for threshold breaches
 Sensor-based data collection and tracking
 Historical data analysis and trend insights
 Admin panel for managing sensors, users, and data
 
 Objective
To provide a centralized and intelligent system for monitoring environmental pollution, promoting awareness, and supporting timely action for a cleaner and safer ecosystem.
